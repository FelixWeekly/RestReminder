# Rest Reminder

一个轻量的 Windows 休息提醒工具（Python + Tkinter）。

## 功能

- 默认每 20 分钟进行一次眼睛休息提醒。
- 默认每 2 小时进行一次强制休息提醒。
- 可自定义小提醒间隔、大提醒间隔和强制休息时长。
- 启动后会保存配置到 `config.json`。
- 主界面为 Quiet Light 风格，整体字号与窗口更大。
- 主界面重做为 Quiet Light 风格，大字号、圆角卡片、布局更自然。
- 强制休息为居中弹窗，交互更自然。
- 小提醒改为应用内浮动提示（非阻塞），不再依赖系统通知中心。

## 测试提醒

- 点击“立即测试”展示提醒窗口。

## 使用建议

- 小提醒建议 15~30 分钟。
- 大提醒建议 90~180 分钟。
- 强制休息时长建议 60~300 秒。
